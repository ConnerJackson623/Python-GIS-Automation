# Python GIS Automation

## Overview

This repository contains introductory Python programming exercises completed in ArcGIS Pro and Python. These exercises demonstrate automation techniques used in GIS workflows, including iterating through feature classes, counting spatial features, and using loops for data processing.

## Skills Demonstrated

- Python
- ArcPy
- ArcGIS Pro
- For Loops
- Feature Class Management
- GIS Automation
- Data Processing
